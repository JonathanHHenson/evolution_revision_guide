# Timestamped transcript index

This is a curated route through the complete English auto-caption tracks for the
eight recorded lessons. It is not a replacement for the captions and it does not
pretend that an auto-caption is a verbatim, speaker-labelled transcript. Each
**Watch** link opens the relevant livestream at the point where the indexed
explanation begins; each **Revision note** opens the short lesson file that
reconstructs the surrounding argument in detail.

The index covers Will's opening presentations, Erika's teaching and Will's
questions during the lesson. The audience/superchat Q&A at the end of each stream
is outside the guide's scope. For exact quotation, names or disputed wording,
listen to the recording rather than quoting the auto-caption text.

## Caption provenance and reproducibility

The timestamp basis is the English `en-orig` YouTube automatic-caption track
downloaded on 15 July 2026. YouTube may later regenerate a track or adjust a
video, so a cue can move by a few seconds. The linked timestamp is the stable
study reference; the recording remains the authority.

To retrieve an automatic-caption track without downloading video media, use a
current version of `yt-dlp`:

```bash
yt-dlp --skip-download --write-auto-subs --sub-langs 'en.*' --sub-format vtt \
  'https://www.youtube.com/watch?v=VIDEO_ID'
```

Do not commit downloaded caption tracks without permission to redistribute them.
They are generated source material, can change and make the repository much
larger. See the [livestream catalogue](livestream-catalog.md) for video metadata
and [further reading](further-reading.md) for the external evidence trail.
